FitTrack

FitTrack is a Flutter-based fitness app designed to help users track and manage their workout plans efficiently. This project focuses on workout plan management, user authentication with Firebase, and a clean, minimal UI.

Features
--------
- User Authentication with Firebase (Email/Password)
- Browse and manage workout plans
- Create, update, and delete custom workout plans
- Real-time synchronization with Firebase Firestore
- Responsive UI optimized for mobile devices

Setup Instructions
------------------
1. Clone the repository
   git clone https://github.com/yourusername/fittrack.git
   cd fittrack

2. Install Flutter dependencies
   flutter pub get

3. Add Firebase configuration files
   - For Android: place google-services.json in android/app/
   - For iOS: place GoogleService-Info.plist in ios/Runner/

4. Run the app
   flutter run
